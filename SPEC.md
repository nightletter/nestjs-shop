### 기술스택
pnpm, nestjs, typeorm 

### 핵심기능

1. 로그인
- JWT 기반 로그인
- 가입된 정보가 없을 시 회원가입 창에서 회원가입 가능 (ID, 비밀번호만 입력해서) -> 가입 정보는 아이디, 비밀번호만

2. 상품
- 카테고리 (아우터, 상의, 하의) 별 상품 목록 조회
- 상품 상세 조회
- 사이즈 옵션 필요 (s, m, l, xl, xxl)
- 소재, 원산지, 세탁방법 등 상품 정보
- 판매가격, 세일가격 필요
- 장바구니에 추가 기능

3. 장바구니
- 조회, 수량변경, 삭제 기능 필요
